# MailChimp Newsletter Signup
Newsletter signup built using Mailchimp <br/> 
Fill your {API-key} and {list-id} in app.js <br/>
  
# Quick Start 
#Install dependencies<br/>
npm install <br/><br/>
#Run server on port 3000 <br/>
npm start <br/>
